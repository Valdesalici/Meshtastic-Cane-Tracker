# 3D Case for Heltec T114 — Dog Harness Edition

This folder contains the 3D printable case for the Heltec T114 V2, modified for attachment to a dog's H-style harness.

## Files

- `case_heltec_t114_canetto.stl` — Main case file with 4 attachment loops for H-style dog harness mounting

## Modifications

This version includes the following changes to the original H2T case:

- **Added 4 attachment loops** (rings) — two on each side of the case
- The loops are designed to accept **2 strips** (zip ties or hook-and-loop straps) that pass through opposite loops
- Enables secure attachment to the **dorsal (back) section of an H-style dog harness**
- The two strips wrap around the harness straps, keeping the tracker firmly in place without interfering with the dog's movement

| Modification | Purpose |
|--------------|---------|
| 4 attachment loops | Pass-through points for mounting straps |
| Two-strip system | Even weight distribution, stable attachment |
| H-style harness compatibility | Designed for the dorsal strap common in H-style harnesses |

## Attachment Method

The case attaches to the harness using **two strips** (zip ties or hook-and-loop straps):

1. Pass the first strip through the two loops on one side of the case
2. Wrap it around the harness's dorsal strap and secure
3. Repeat with the second strip on the opposite side
4. Ensure both strips are tight enough to prevent sliding, but not so tight that they deform the case

This method keeps the tracker centered on the dog's back, with minimal interference during movement.

## Printing Recommendations

| Setting | Value |
|---------|-------|
| **Material** | PETG or ASA (weather resistant, UV resistant) |
| **Layer height** | 0.2 mm |
| **Infill** | 30% |
| **Supports** | Not required (print face down) |
| **Orientation** | Print with the display opening facing down on the build plate |

The attachment loops are designed to print without supports if oriented correctly.

## Assembly Notes

1. Remove the clear plastic frame from the display
2. Install buttons, board, GPS module, and SMA pigtail
3. Snap the two halves together
4. Secure with an M3 screw
5. **Before final assembly**, pass the two mounting strips through the loops if using non-releasable zip ties

## Recommended Components

- **Heltec T114 V2** with 1.14" display
- **L76K GPS module**
- **LiPo battery**: up to 8mm × 31mm × 60mm
- **SMA pigtail**: U.FL to SMA, 10cm
- **Antenna**: SMA, up to 12.7mm diameter at the base
- **M3 screw** for final closure
- **Mounting strips**: 2x zip ties (3.6mm width recommended) or hook-and-loop straps
